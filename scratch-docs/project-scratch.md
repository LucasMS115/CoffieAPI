# ☕ Coffee App — Rascunho de Funcionalidades

## 🔹 Core Features

### 1. Base Coffee

* **1.1 Salvar receitas**
* **1.2 Buscar um café / criar um café**
* **1.3 Inputs da receita**
* **1.4 Avaliação (com notas e reviews)**

---

### 2. Perfil

#### 2.1 Histórico de receitas

* Estatísticas:
  * Avaliações 
  * Preferências (gerada automaticamente)
  * Informações gerais (inclusas pelo usuário)

#### 2.2 Acesso a receitas de outros usuários

* acesso completo
* receitas podem ser privadas por usuários autores


---

### 3. Home

* **3.1 Minhas receitas**
* **3.2 Criar receita**
* **3.3 Buscar receita**
* **3.4 Explorar cafés**
  * Leva para:
    * Coffie Hub

---

### 4. Coffie Hub

Página principal com entrada para as páginas a seguir

#### 📄 Coffie Hub - Páginas dos Cafés

* Página gerada com:
  * Notas
  * Receitas
  * Comentários
  * Avaliações
  * Links de compra

#### Coffie Hub - Outras páginas relacionadas:

* Página da marca (com relação de cafés)
* Página de tipos (com relação de cafés):
  * Doce
  * Frutado
  * Fermentado
  * etc.

---

#### Coffie Hub - Página de Método

* Receitas bem avaliadas
* Ultimas receitas
* Relação de cafés mais populares
* Relação de cafés mais bem avaliados


---

## 💡 Ideias Extras

* Botão de captura por assinatura
